LCD12864实验步骤：
1、开发板工作电源选择为5V(即J3端子选择5V)，J9端子选择为PSB_H，J8端子全部使用短路帽短接，J23端子的指示灯短路帽去掉。
2、将12864按照正确方向插到开发板J7上面。
3、MINI USB连接线给开发板通电下载程序，下载软件中内部IRC时钟选择11.0592MHZ。
4、下载程序后，LCD屏上会依次显示内容：
1）在屏第一行显示：LCD12864检测成功。
2）在屏第二行显示：IAP15W4K58S4。
3）在屏第三行显示：液晶屏变量测试：。
4）在屏第四行显示：temp = x 。（x是从0到9不断变化的数字）

注：如果LCD屏没有显示，可使用螺丝刀拨动开发板上可调电阻RP1调节显示灰度。
